mod-spdy
========

OpenSSL 1.0.1(f) and Apache 2.4.7 port for mod-ssl with npn support and mod-spdy. Tested under Ubuntu 14.04.

Required Packages
=================

$ sudo apt-get install subversion curl g++ apache2 libapr1-dev libaprutil1-dev patch binutils make devscripts
